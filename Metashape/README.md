# Metashape placeholder

This directory is intentionally kept without the Agisoft Metashape binaries.

For local development or private packaging, either:

- install Metashape and expose `metashape.exe` through `PANOFUSION_METASHAPE`, or
- place the portable executable at `Metashape/App/Metashape/metashape.exe`.

Do not commit the Metashape application files to the public repository.
